## ✅ **八宫六十四卦对照表（含六亲、世应、伏神规则）**

> **依据**：京房八宫卦变体系（《京氏易传》）\
> **核心规则**：
>
> * 每宫8卦，共64卦。
> * 本宫纯卦（首卦）**世爻在上爻（六爻）**。
> * 世爻位置按“游魂—归魂”规律变化。
> * 六亲以**本宫五行**为基准，依生克关系定。
> * 伏神：本宫卦五行所藏之爻，用于补缺用神。

***

### 📌 **通用规则说明**

| 项目 | 规则 |
|------|------|
| **本宫五行** | 决定六亲归属的基准 |
| **兄弟** | 与本宫五行相同者 |
| **父母** | 生本宫五行者（印星） |
| **子孙** | 本宫五行所生者（食伤） |
| **妻财** | 克本宫五行者（财星） |
| **官鬼** | 克本宫五行者（官杀） |
| **世爻位置** | 本宫卦在六爻，后按“一、二、三、四、五、游、归”排 |
| **应爻位置** | 与世爻相隔两爻（初↔四，二↔五，三↔上） |
| **伏神** | 本宫五行所缺之爻，伏于飞爻之下，以本宫五行所生或同类补之 |
